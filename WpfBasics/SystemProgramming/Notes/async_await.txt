async / await
Ключевые слова async / await используются для формализации работы с 
многозадачными приложениями.
Главная особенность - подчеркнуть тот факт, что запуск метода
происходит в асинхронном режиме.

Синхронный и асинх. режимы принято разделять, поэтому
а) вызов (await) асинхронный методов разрешено только в других асинхронных методах
б) методы без асинхронной части не рекомендуется объявлять как async

вариант 1 (не нить)
Window           Task
loader  ---------> { FileContentAsync
loading...         |
                   |
Content <----------}
dic = await ----------->{ ParseIniAsync
                        |
                        |
dic    <----------------}
LogDic(dic) ----->{ LogDicAsync
}                 |
                  |
                  }


вариант 2 (нить)
Window           Task
task  ---------> { FileContentAsync
loading...       |
}                | return
                 } -----> { ParseIniAsync
                          |
                          | return
                          } ------> { LogDicAsync
                                    |
                                    |
                                    }
